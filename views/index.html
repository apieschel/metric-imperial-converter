Metric-Imperial Converter

  1. Send an accepted number and unit in a single parameter to GET /api/convert to have it converted.
  2. "Gal" converts to "L" and vice versa. (1 gal to 3.78541 L)
  3. "Lbs" converts to "kg" and vice versa. (1 lbs to 0.453592 kg)
  4. "Mi" converts to "km" and vice versa. (1 mi to 1.60934 km)
  5. If unit of measurement is invalid, "invalid unit" is returned.
  6. If number is invalid, "invalid number" is returned.
  7. If both are invalid, "invalid number and unit" is returned.
  8. Fractions, decimals, and a combination of the two are acceptable (ie. 5, 1/2, 2.5/6).
  9. If no parameter is provided, the measurement will default to 1.
  10. The standard returned object consists of initNum, initUnit, returnNum, returnUnit, and string spelling out units in the following format:
    {initNum} {initial_Units} converts to {returnNum} {return_Units}
  11. Results are rounded to 5 decimal places.

Example usage:

/api/convert?input=4gal
/api/convert?input=1/2km
/api/convert?input=5.4/3lbs
/api/convert?input=kg

Example return:

{initNum: 3.1, initUnit: 'mi', returnNum: 5.0000008, returnUnit: 'km', string: '3.1 miles converts to 5.00002 kilometers'}
